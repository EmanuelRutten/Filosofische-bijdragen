====================================================
Filosofische bijdragen — Emanuel Rutten
====================================================

Repository:
https://github.com/EmanuelRutten/Filosofische-bijdragen

DESCRIPTION
-----------
“Filosofische bijdragen” is a curated collection of philosophical essays,
articles, and lectures authored by Emanuel Rutten. The repository functions
as an open digital archive of his philosophical work, covering topics such
as philosophy of religion, metaphysics, epistemology, logic, rhetoric,
and cultural criticism.

The repository is designed to be browsed primarily via an HTML index
rather than as a conventional code project.

INDEX.HTML (STARTING POINT)
---------------------------
The file `index.html` is the central entry point of this repository.

It provides a structured overview of the available philosophical
contributions and acts as a navigational hub linking to the individual
documents (mostly PDF files). Users are encouraged to open `index.html`
in a web browser to explore the collection in an organized and readable
way.

When hosted (e.g. via GitHub Pages or locally), `index.html` functions
as a lightweight website presenting the contents of this archive.

CONTENTS
--------
The repository consists primarily of standalone philosophical texts,
mostly in PDF format. Each file corresponds to a specific contribution
by Emanuel Rutten.

Examples include:
    • AMetaphysicalPrincipleEntailingTheism_ERutten.pdf
    • AModalEpistemicArgumentForTheExistenceOfGod_ERutten.pdf
    • Aristoteles Metafysica en Organon.pdf
    • ChatGPTendemens_3_ERutten.pdf
    • DeKleineRetorica_ERutten.pdf
    • HeideggersZijn_ERutten.pdf
    • ModaalEpistemischArgument_ERutten.pdf

The filenames generally indicate the subject matter.
The complete and up-to-date list of works is presented in `index.html`.

HOW TO USE THIS REPOSITORY
--------------------------
1. Open `index.html` in a web browser for an overview of the collection.
2. Click links in the index to access individual essays and articles.
3. Download PDFs for offline reading, study, or academic reference.
4. The repository contains no executable software; it is an archive
   of philosophical texts.
